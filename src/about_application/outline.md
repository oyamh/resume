# 概要
- ブックマークをインポートして管理するWebサービスです。
- 多くを外部ライブラリに依存せずに自力で実装しました。
    - React,Yew,gRPCフレームワークなどの基幹部分を除いて。
- 検索クエリの組み合わせによって柔軟な検索ができます。
- 補完入力で簡単に検索できます。
- マークダウンでメモを追加して表示＆検索ができます。
